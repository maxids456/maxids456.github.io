Поддерживаемые клиенты:

🔧 Способ для HAPP и похожих (рекомендуем):

  1. Найди кнопку «Добавить подписку» или «Import from URL»
  2. Вставь скопированную выше ссылку в поле URL и нажми → «ОК»
  3. Подключайся к серверам и открывай доступ ко всему интернету!

🤖 Добавление серверов файлом (запасной):

  1. Нажми «Скачать файлом» (кнопка выше), чтобы получить MaxBar.txt
  2. В приложении нажми «➕» → «Import Config from File» и выбери скачанный файл
  3. Выбирай нужный сервер и пользуйся незаблокированным интернетом! 🥳
⚠️ Важно: При добавлении подписки через ссылку (скопированную выше) автообновление работает (сервера сами обновляются при изменении)
При добавлении через файл автообновления НЕТ — нужно будет периодически скачивать новый файл
