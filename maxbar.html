🔧 Универсальный способ (для любого прокси-клиента):

  1. Открой свой прокси-клиент (HAPP, Shadowrocket, v2rayNG, v2rayTun, Hiddify и др.)
  2. Найди кнопку «Добавить подписку» или «Import from URL»
  3. Вставь скопированную выше ссылку в поле URL → «ОК»
  4. Если клиент не поддерживает автоматический импорт, скачай MaxBarVPN.txt и добавь как файл подписки.
⚠️ Важно: При добавлении подписки через ссылку (скопированную выше) автообновление работает (сервера обновляются при изменении файла на сайте).
При добавлении через скачанный файл .txt автообновления НЕТ — нужно будет периодически скачивать новый файл и добавлять его заново.

⚡ Для HAPP:

  • Открой HAPP → нажми «➕ Добавить подписку» → вставь ссылку → «ОК»

🤖 Для v2rayNG / v2rayTun:

  • Нажми «➕» → «Import Subscription» → вставь ссылку → «OK»
    Или используй скачанный файл: «➕» → «Import Config from File»

🌀 Для Hiddify:

  • Нажми «➕» → «Новая подписка» → вставь ссылку → «Добавить»
🔒 Подписка обновляется автоматически (сервера в одном месте)
