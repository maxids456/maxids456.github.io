🔧 Универсальный способ (для любого прокси-клиента):

  1. Открой свой прокси-клиент (HAPP, Shadowrocket, v2rayNG и др.)
  2. Найди кнопку «Добавить подписку» или «Import from URL»
  3. Вставь скопированную выше ссылку в поле URL
  4. Нажми «Добавить» → сервера появятся в списке
  5. Подключайся и наслаждайся!

⚡ Для HAPP:

  • Открой HAPP → нажми «➕ Добавить подписку» → вставь ссылку → «ОК»

🍎 Для Shadowrocket:

  • Нажми «+» → выбери «Subscribe» → вставь ссылку → «Save»

🤖 Для v2rayNG:

  • Нажми «➕» → «Import Subscription» → вставь ссылку → «OK»
